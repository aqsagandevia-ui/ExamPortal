# Online Examination Portal

## Project Description

The **Online Examination Portal** is a web-based application designed to conduct online exams efficiently. It allows students to take exams remotely while enabling administrators to manage questions, users, and results in a structured manner.

---

## Features

### Student Panel

* User Registration & Login
* Attempt Exams
* View Results & Scores
* Timer-based Exam System

### Admin Panel

* Add/Edit/Delete Questions
* Manage Students
* Create Exams
* View Results & Analytics

## Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/aqsagandevia-ui/ExamPortal.git
```

### 2️⃣ Navigate to Project

```
cd ExamPortal
```

### 3️⃣ Install Dependencies

```
npm install
```

### 4️⃣ Run Project

```
npm run dev
```

### 5️⃣ Open in Browser

```
http://localhost:5173
```
